# Writely-AI-Writing-Assistant
Developed Writely which is an AI based web-app that assists users in writing. It can help users to correct spelling, grammar and rephrase sentences to improve quality of writing.

## About Writely

Writely is an AI-powered writing assistant designed to help users improve their writing through intelligent spelling correction, grammar checking, and sentence rephrasing. The application leverages OpenAI APIs and GPT-based language models to understand context, generate accurate corrections, and provide natural-sounding rephrased text.

By using advanced AI and natural language processing capabilities, Writely goes beyond basic rule-based checks and delivers context-aware suggestions that enhance clarity, tone, and overall writing quality. The system processes user input in real time, making it a reliable tool for drafting, editing, and refining written content.

## Features

- AI-powered spelling correction. 
- Grammar checking and correction.
- Sentence rephrasing for improved clarity and tone.
- Real-time text processing. 
- Secure user authentication using Privy.  
- Clean and responsive UI built with Tailwind CSS.


## Tech Stack

- **React** – Frontend library for building user interfaces.
- **Vite** – Fast development build tool and development server.
- **Tailwind CSS** – Utility-first CSS framework for styling. 
- **Node.js** – JavaScript runtime environment used to build the backend server, handle API requests, process AI-powered writing features, and manage frontend-backend communication.
- **Privy** – Authentication and user management.



## 1. Steps to run the web-app

**a. Server**

* Open folder server in your integrated terminal using `cd server`.
* Run command `npm run start` OR `node app.js` to start server for backend.

**b. Client**

* In a new terminal, open folder client, using `cd client`.
* Start the development server using `npm run dev`.
* This project uses Vite for development. After starting the dev server, the app will be available at local development server URL.
